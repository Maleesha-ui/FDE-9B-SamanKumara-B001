## Appendix B: Query Reference

### B.1 Completeness Query
```sql
SELECT 
    COUNT(*) AS source_count,
    SUM(CASE WHEN target_id IS NOT NULL THEN 1 ELSE 0 END) AS matched_count,
    SUM(CASE WHEN target_id IS NULL THEN 1 ELSE 0 END) AS missing_count,
    (SUM(CASE WHEN target_id IS NOT NULL THEN 1 ELSE 0 END) / COUNT(*)) * 100 AS completeness_pct
FROM source_records s
LEFT JOIN target_records t ON s.id = t.source_id
WHERE s.batch_id = 'BATCH-GL-20260328-0930';
B.2 Accuracy Query
sql
WITH field_comparison AS (
    SELECT 
        s.id,
        CASE WHEN s.document_id = t.document_id THEN 1 ELSE 0 END AS doc_match,
        CASE WHEN s.amount = t.amount THEN 1 ELSE 0 END AS amount_match
    FROM source_records s
    JOIN target_records t ON s.id = t.source_id
    WHERE s.batch_id = 'BATCH-GL-20260328-0930'
)
SELECT 
    COUNT(*) AS total_records,
    SUM(doc_match) AS doc_matches,
    SUM(amount_match) AS amount_matches,
    (SUM(doc_match + amount_match) / (COUNT(*) * 2)) * 100 AS accuracy_pct
FROM field_comparison;
B.3 Timeliness Query
sql
SELECT 
    COUNT(*) AS total_records,
    AVG(EXTRACT(EPOCH FROM (t.loaded_at - s.extracted_at))) AS avg_latency_seconds,
    SUM(CASE 
        WHEN (t.loaded_at - s.extracted_at) <= INTERVAL '4 hours' 
        THEN 1 ELSE 0 
    END) AS within_sla,
    (SUM(CASE 
        WHEN (t.loaded_at - s.extracted_at) <= INTERVAL '4 hours' 
        THEN 1 ELSE 0 
    END) / COUNT(*)) * 100 AS timeliness_pct
FROM source_records s
JOIN target_records t ON s.id = t.source_id
WHERE s.batch_id = 'BATCH-GL-20260328-0930';
Appendix C: Glossary
Term	Definition
Batch ID	Unique identifier for each integration batch
DLQ	Dead Letter Queue - Failed records awaiting review
Checksum	Mathematical sum used for data validation
Variance	Difference between source and target totals
Throughput	Number of records processed per second
Latency	Time delay between source and target
SLA	Service Level Agreement - Performance targets
Reconciliation	Process of comparing source and target data
Zero-Break	Reconciliation with zero breaks or variances
Completeness	Percentage of records successfully transferred
Accuracy	Percentage of fields correctly transferred
Timeliness	Percentage of records within SLA
Consistency	Percentage of records with no logical contradictions
