# FDE-9B: SAP S/4HANA to FinSight Integration

**Project:** Custom API Integration between Client's ERP and Financial Analytics Platform
**Student:** Maleesha Ransarani

**Start Date:** 2026-06-28

## Project Overview

This project designs and implements a production-grade API integration framework connecting Meridian Manufacturing's SAP S/4HANA ERP system with Zetheta FinSight financial analytics platform.

## Client Context

* **Company:** Meridian Manufacturing Ltd.
* **Revenue:** INR 2,400 Crore
* **Plants:** 7 locations (Maharashtra, Gujarat, Tamil Nadu)
* **ERP:** SAP S/4HANA 2023 FPS02
* **Current State:** 24-hour batch delay in financial reporting
* **Target State:** Real-time (4-hour SLA) financial analytics
